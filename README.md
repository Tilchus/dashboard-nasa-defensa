# 🗺️ Dashboard NASA: Inteligencia de Defensa Planetaria

Bienvenido al centro de monitoreo táctico. Esta aplicación web interactiva está desarrollada sobre **R Shiny** y utiliza la infraestructura de datos abiertos de la **NASA** para auditar, visualizar y analizar las amenazas espaciales en tiempo real.

### ⚡ Características Principales:
* **Centro de Control NEO:** Escaneo y filtrado dinámico de Objetos Cercanos a la Tierra según su fecha de aproximación.
* **Matriz de Riesgo Cinético:** Gráficos analíticos e interactivos que cruzan el diámetro estimado frente a la velocidad relativa de los cuerpos celestes.
* **Registro Automatizado:** Tablas inteligentes optimizadas mediante Flexbox y JavaScript para un seguimiento fluido de avistamientos.
* **Exploración APOD:** Acceso al archivo multimedia diario de la NASA con un algoritmo elástico de traducción inglés-español integrado.

---
_Nota: Diseñado con una arquitectura modular responsiva utilizando Bootstrap 5 (Bootswatch Darkly), optimizado tanto para monitores de escritorio como para visualizadores integrados._
